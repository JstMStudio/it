Crea repository

Forniamo diversi metodi per consentire a tutti di avviare e creare il proprio repository su Jst multiuso posto di lavoro. Gli utenti possono scegliere un metodo per tutti i metodi forniti qui per avviare e creare un repository.



Metodi


Puoi creare un problema nel nostro repository di documentazione nell'organizzazione Jst multiuso posto di lavoro Github. Potresti creare un problema indicando tutte le informazioni sul tuo repository e così via, in modo che un volontario possa intervenire e creare il repository per te.

Puoi creare un repository creando un ticket per contattarci. Puoi contattarci e comunicarci le informazioni che desideri nel tuo repository in modo che lo staff di Jst multiuso posto di lavoro possa creare un repository per te. Questo funzionerà solo durante l'orario di lavoro del personale di Jst multiuso posto di lavoro. Puoi creare un ticket sul nostro Discord/Support Desk.

Non solo, potresti anche riempire il modulo Google del repository Jst multiuso posto di lavoro. Potresti compilare il modulo Google con tutte le informazioni nei campi necessari. Al termine della compilazione, invia il modulo a Jst multiuso posto di lavoro e attendi che un creatore del repository o il personale OFUT crei un repository per te, operazione che di solito richiedeva da poche ore a giorni.
